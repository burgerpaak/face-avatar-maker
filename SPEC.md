# 페이스 아바타 메이커 — Spec

## 개요
얼굴 전용 프로필 아바타 커스터마이저. 픽셀아트 SVG 기반 (10x10px 그리드, 240x240 캔버스)으로, 이목구비/헤어/액세서리를 조합해 나만의 아바타를 만든다.

---

## 디자인 시스템

### 컬러
- 라이트모드 블랙 & 화이트
  - `--bg: #f5f5f5` / `--surface: #ffffff` / `--border: #e2e2e2`
  - `--text: #111` / `--text2: #777`
- 아바타 렌더링: 픽셀아트 스타일
  - 얼굴 외곽선: `#000` (검정)
  - 눈/입: `#000` (검정)
  - 머리카락 기본: `#666`
  - 피부/헤어/액세서리: 컬러 팔레트로 변경 가능

### 타이포그래피
- 폰트: Outfit + Noto Sans KR
- 텍스트 두께: 전체 기본(400~500), 볼드 사용 안 함
- 사이드바 타이틀: 16px / 500
- 사이드바 메뉴: 15px / 400
- 서브사이드바 타이틀: 15px / 400
- 옵션 라벨: 11px / 500
- 옵션 버튼: 13px (→ 추후 이미지로 교체 예정)

### 인터랙션
- 호버: `#f7f7f7` (연한 그레이)
- 선택: `#f0f0f0` (호버보다 약간 진한 그레이)
- 텍스트 두께 변화 없음 (호버/선택 시 동일)
- `user-select: none` (사이드바, 서브사이드바)

## 레이아웃

### 구조: 3단 패널 (100vh 고정)
```
[사이드바] | [서브사이드바(옵션)] | [아바타 미리보기]
```

### 사이드바 (좌측)
- 넓이: `clamp(170px, 13vw, 210px)`
- 타이틀: "아바타 메이커" (하단 라인 없음)
- 카테고리 메뉴: 상하 중앙 정렬, 좌측 정렬
- 아이콘: 이모지, 아웃라인/배경 없음
- 하단 고정: 💾 다운로드 버튼 (아웃라인 스타일)

### 서브사이드바 (중간)
- 넓이: `clamp(220px, 24vw, 360px)`
- 첫 화면에서 닫힌 상태 (`width: 0`, 트랜지션 애니메이션)
- 헤더: 카테고리 이름 (텍스트만, 아이콘 없음) + 🎲 랜덤 버튼 (우측 끝)
- 옵션 박스: 정방형(`aspect-ratio: 1/1`), 3열 그리드
- 옵션 스타일: `border-radius: 12px`, 기본 투명+아웃라인, 호버/선택 시 fill
- 내부 카테고리 라벨: 없음 (제거됨)
- 랜덤: 현재 선택된 카테고리의 옵션만 랜덤 변경

### 아바타 미리보기 (우측)
- 나머지 영역 전체 (`flex: 1`)
- 카드: 원형 (`border-radius: 50%`, `aspect-ratio: 1/1`)
- 카드 크기: `min(560px, calc(100vh - 20px))`
- 카드 shadow: `0 4px 40px rgba(0,0,0,0.05)`
- 내부 SVG: width/height: 88%, `viewBox: 0 0 240 240`
- 목(neck) 없음, 얼굴만

## 카테고리 & 옵션

| 카테고리 | 아이콘 | 옵션 | 컬러 |
|---------|-------|------|------|
| 얼굴 | 👤 | 5종 | ✅ 피부 컬러 팔레트 6색 |
| 눈 | 👁 | 13종 | - |
| 입 | 👄 | 9종 | - |
| 헤어스타일 | 💇 | 18종 (앞머리 + 뒷머리) | ✅ 컬러 팔레트 |
| 액세서리 | ✨ | 16종 | ✅ 컬러 팔레트 |

### 컬러 팔레트
- 위치: 서브사이드바 옵션 그리드 하단
- UI: 원형 컬러 칩 가로 나열
- 적용 대상: 얼굴(피부), 헤어, 액세서리
- 피부 컬러: `#fff`, `#e6e6e6`, `#b3b3b3`, `gray`, `#4d4d4d`, `blue`
- 헤어 컬러: `#666` (기본) + 추가 컬러
- 액세서리 컬러: `blue` (기본) + 추가 컬러

### 렌더링 레이어 순서
1. 헤어 뒷부분 (hairBack)
2. 피부 채움 (얼굴형 내부를 flood fill로 계산, 선택한 컬러 적용)
3. 얼굴형 (검정 외곽선)
4. 눈
5. 입
6. 헤어 앞부분
7. 액세서리

### 피부 렌더링 방식
- 얼굴형 SVG의 외곽선(rect 좌표)을 분석하여 내부 영역을 자동 계산 (flood fill)
- 선택한 피부 컬러로 내부 픽셀을 채움
- 별도 피부 SVG 파일 불필요 (컬러 선택만)

## 기능

### 첫 화면
- 아바타 카드 위에 버튼 2개 오버레이 (카드 중앙 아래쪽)
  - 🎲 (원형, 56px) — 전체 카테고리 랜덤, 서브사이드바 안 열림, 반복 클릭 가능
  - ✏️ 새로 만들기 (pill) — 얼굴형만 첫 번째 옵션 선택, 나머지 초기화, 서브사이드바 오픈
- 사이드바 메뉴 클릭 시에도 오버레이 사라짐

### Undo / Redo
- 랜덤 버튼(서브사이드바 내) 클릭 시 히스토리 저장
- 우하단에 통합 pill 박스 (↩ | ↪)
- 히스토리 없으면 숨김, 끝 도달 시 해당 버튼 비활성화 (`opacity: 0.25`)

### 다운로드
- 사이드바 하단 고정 버튼
- PNG 560x640 출력, 배경 `#f5f5f5`

### 리소스
- 아바타 파츠: 픽셀아트 SVG (10x10px rect 기반, viewBox 0 0 240 240)
- 일러스트레이터에서 제작, 개별 SVG 파일로 내보내기
- SVG 옵션: Styling=Inline Style, Minify, Responsive
- `assets/` 폴더에 카테고리별 정리
- 코드 내 인라인으로 SVG 데이터 임베드 (단일 HTML 유지)
- 피부: 별도 SVG 없음 (얼굴형 기반 flood fill + 컬러 팔레트)

```
assets/
├── face/         5개 (외곽선만, 검정)
├── eyes/         10개 (파츠만, 검정)
├── mouth/        9개 (파츠만, 검정)
├── hair/         7개 (앞머리, #666, 뒷머리 추후)
└── accessories/  16개 (blue 기본)
```

## 반응형 (모바일)
- 820px 이하: 세로 스택
- 사이드바 → 가로 스크롤 메뉴
- 서브사이드바 → 최대 높이 220px
- 아바타 카드 → 260px
